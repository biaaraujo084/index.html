SOBRE
Html HTML é a linguagem padrão para criar páginas da web. Ele usa tags para definir a estrutura e o conteúdo de uma página. As tags são elementos entre < e >, como

para parágrafos e para imagens. HTML também inclui elementos para formulários, listas e mídia. Desde HTML5, há um foco crescente na semântica, usando tags como "header" e "footer" para descrever a estrutura do conteúdo.
JavaScript JavaScript é uma linguagem de programação usada para tornar páginas web interativas. Ela é executada no navegador do usuário e permite manipular elementos HTML, responder a eventos e controlar o comportamento da página. JavaScript também pode ser usado no lado do servidor com Node.js.
Arduino O Arduino é uma plataforma de prototipagem eletrônica de código aberto que consiste em hardware e software fácil de usar. Ele permite que pessoas, desde iniciantes até especialistas, criem projetos interativos que envolvem sensores, atuadores e outras componentes eletrônicas. O coração do Arduino é uma placa com um microcontrolador programável, que pode ser programado usando uma linguagem baseada em C/C++. Com uma vasta comunidade de usuários e uma grande variedade de componentes e shields disponíveis, o Arduino é amplamente utilizado em projetos de hobby, educação e prototipagem rápida.
Levantamento de Requisitos O levantamento de requisitos é o processo de entender e documentar as necessidades dos usuários para um sistema de software. Isso envolve coletar informações, analisá-las, validá-las e comunicá-las de forma eficaz aos stakeholders. Essa etapa é fundamental para garantir o sucesso do projeto, evitando erros e garantindo que o sistema atenda às expectativas dos usuários.
CCS CSS é uma linguagem usada para estilizar páginas da web. Ela controla a aparência visual dos elementos HTML, como cor, fonte e layout. Com CSS, é possível criar estilos flexíveis e reutilizáveis para páginas da web.
VisualG O VisualG é uma ferramenta de programação que usa pseudocódigo para ajudar estudantes a aprenderem programação de forma visual e interativa. Ele oferece uma interface simples e passo a passo para escrever e executar algoritmos, sendo amplamente utilizado em ambientes educacionais.
Sistemas Operacionais Um sistema operacional é um software que coordena recursos de computadores e fornece uma interface para os usuários. Ele gerencia processos, memória, armazenamento e dispositivos de entrada/saída. Em suma, é o "cérebro" por trás de como um computador funciona.
